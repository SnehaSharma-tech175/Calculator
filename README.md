# Calculator Using Python Tkinter

## Introduction

This project is a simple calculator application developed using **Python** and the **Tkinter** library. It provides a graphical user interface (GUI) to perform basic arithmetic calculations. The project was created to understand GUI development and event handling in Python.

## Features

* Simple and easy-to-use interface
* Performs addition, subtraction, multiplication, and division
* Supports decimal numbers
* Clear (C) button to reset the input
* Delete (DEL) button to remove the last entered character
* Displays **Error** for invalid expressions

## Technologies Used

* Python 3
* Tkinter (Python GUI Library)

## How to Run the Project

1. Install Python 3 on your computer.
2. Save the project file as `calc_sample.py`.
3. Open a terminal or command prompt.
4. Run the following command:

```bash
python calc_sample.py
```

The calculator window will open, and you can start performing calculations.

## Working

* Click the number and operator buttons to enter an expression.
* Press the **=** button to calculate the result.
* Press **C** to clear the display.
* Press **DEL** to delete the last entered character.
* If an invalid expression is entered, the calculator displays **Error**.

## Learning Outcomes

Through this project, I learned:

* Basics of Python GUI programming using Tkinter
* Creating and arranging widgets
* Handling button click events
* Using functions in Python
* Building a simple interactive application

## Conclusion

This calculator project demonstrates the basics of GUI application development in Python. It is a beginner-friendly project that helps in understanding event-driven programming and can be extended with additional features such as scientific calculations, percentage, square root, and calculation history.
